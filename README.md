# Vietnam E-Commerce Storytelling

Data storytelling and visualization project analyzing Vietnam E-commerce trends in 2023 using Python and data visualization libraries.

---

## Project Overview

This project focuses on exploring and visualizing Vietnam E-commerce data to uncover customer behavior, delivery performance, regional differences, and demographic shopping trends.

Using storytelling techniques combined with data visualization, the project transforms raw transactional data into meaningful business insights that support better decision-making.

---

## Dataset

### Dataset File
- `vietnam_ecommerce_2023.csv`

### Dataset Description
The dataset contains simulated Vietnam E-commerce transaction records from 2023, including information related to:
- Customer demographics
- Delivery times
- Customer ratings
- Sales channels
- Regional activity
- Purchasing behavior

---

## Project Objectives

The main goals of this project are:

- Analyze how delivery performance impacts customer satisfaction
- Compare E-commerce trends across regions in Vietnam
- Explore customer shopping channel preferences
- Understand demographic purchasing patterns
- Create business recommendations from analytical insights
- Apply storytelling techniques to data visualization

### Option 1: Clone Using Git

```bash

```

Move into the project folder:

